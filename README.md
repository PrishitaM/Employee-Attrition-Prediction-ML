# Employee-Attrition-Prediction-ML
Machine Learning project to analyze and predict employee attrition using EDA, data preprocessing and classification models like Logistic Regression, Random Forest, XGBoost, AdaBoost, and Gradient Boosting.
# HR Attrition Analysis & Prediction

## Project Overview
This project focuses on analyzing employee attrition data and building machine learning models to predict whether an employee is likely to leave the organization.

The objective is to identify key factors affecting attrition and provide actionable insights to improve employee retention.

---

## Problem Statement
Employee attrition is a major concern for organizations as it leads to increased hiring costs and productivity loss.  
This project aims to:
- Analyze employee data
- Identify patterns and trends
- Predict attrition using machine learning models

---

## Dataset
The dataset contains employee-related information such as:
- Age
- Job Role
- Salary
- Work Experience
- Department
- Job Satisfaction
- Attrition (Target Variable)

---

## Exploratory Data Analysis (EDA)
Performed detailed analysis including:
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Visualization using plots (histograms, countplots, heatmaps)

Key insights were derived from relationships between features and attrition.

---

## Data Preprocessing
- Handling missing values
- Encoding categorical variables
- Feature scaling
- Feature selection

---

## Machine Learning Models Used
The following models were implemented and compared:

- Logistic Regression
- Decision Tree
- Random Forest
- AdaBoost Classifier
- Gradient Boosting
- XGBoost Classifier

---

## Hyperparameter Tuning
- GridSearchCV was used to optimize model performance
- Best parameters were selected for each model

---

## Model Evaluation
Models were evaluated using:
- Accuracy Score
- Confusion Matrix
- Precision, Recall, F1-score
- ROC-AUC Curve

---

## Final Model Selection
The best-performing model was selected based on:
- Highest accuracy
- Balanced precision and recall
- ROC-AUC performance

---

## Key Insights
- Employees with low job satisfaction are more likely to leave
- Overtime and work-life balance significantly impact attrition
- Salary and job role play an important role in employee retention
